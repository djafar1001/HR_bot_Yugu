Техническое задание на разработку чат-бота для информирования о вакансиях на предприятии:

Описание задачи:

Разработать чат-бота, который будет предоставлять информацию о вакансиях на определенном предприятии.
Бот должен также предоставлять информацию о требованиях к претенденту для каждой вакансии.
Бот должен выводить ссылки на информационные документы, которые расположены на веб-странице предприятия.
Бот должен иметь возможность принимать анкету претендента, которую можно заполнить в интерфейсе Telegram.
Бот должен предоставлять контактные телефоны службы персонала предприятия и давать возможность сделать звонок по этим номерам.
Функциональные требования:

Бот должен иметь команду или кнопку для получения списка доступных вакансий на предприятии.
При выборе конкретной вакансии, бот должен выводить информацию о требованиях к претенденту.
Бот должен иметь команду или кнопку для получения ссылок на информационные документы о вакансиях.
Бот должен предоставлять форму анкеты претендента, в которой можно указать необходимую информацию.
Бот должен сохранять анкеты претендентов для последующей обработки службой персонала предприятия.
Бот должен предоставлять контактные телефоны службы персонала предприятия.
Бот должен иметь функциональность для совершения звонков по контактным телефонам.
Технические требования:

Бот должен быть разработан с использованием языка программирования Python и библиотеки pyTelegramBotAPI.
Для обработки анкет претендентов можно использовать базу данных или другой подходящий механизм хранения данных.
Бот должен иметь возможность взаимодействовать с веб-страницей предприятия для получения ссылок на информационные документы.
Для возможности совершения звонков по контактным телефонам, бот может использовать соответствующие API или функциональность операционной системы.
Дополнительные требования:

Бот должен иметь удобный и интуитивно понятный интерфейс для пользователя.
